# Quick Reference

## The two commands you always need
- **"Start"** — begin or continue today's session (works for fresh days and continuations)
- **"Save"** / **"End"** / **"Wrap up"** — save progress and log the session; safe to use anytime

## During a session
- **"Quick session"** — ~7 min, busy day: vocab + one speed drill only
- **"Let's just review today"** — no new material, review only
- **"I want a new topic"** — push ahead to new grammar
- **"Let's do a roleplay"** — conversation practice

## Typing accents (optional)
Backtick after the vowel: `hablo`` → habló, `aqui`` → aquí, `tu`` → tú. Never required.

## The loop
1. New conversation → **"Start"**
2. Practice as long as you want
3. **"Save"** → **`/clear`**
